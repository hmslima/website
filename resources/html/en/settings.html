Technical details about this website

This website was build in vanilla JavaScript, no framework was used, so I think it will be easy for you to study it if you want.

As you can see, this is practically an SPA (Single Page Application), but you can bookmark a specific page in a specific language through URL parameters:

  • Languages
    • ?lang=en (English. Actually this one is redundant because it is the default)
    • ?lang=ptbr (Brazilian Portuguese)
  • Pages
    • ?page=home (Home page. The default option)
    • ?page=pro (Professional Profile)
    • ?page=projects (Projects)
    • ?page=settings (Settings)
  • Lighting mode
    • ?mode=light (Light mode. Actually this one is redundant because it is the default)
    • ?mode=dark (Dark mode)
  • You can mix these parameters by using the character &:
    • ?page=pro&lang=ptbr (Professional Profile's page in Brazilian Portuguese)
    • ?lang=en&page=projects (Projects' page in English, although specifying "lang" as "en" is redundant)
    • ?page=pro&lang=ptbr&mode=dark (Professional Profile's page in Brazilian Portuguese, and in dark mode)
