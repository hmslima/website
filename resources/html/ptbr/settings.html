Detalhes técnicos sobre este site

Este site foi construído em JavaScript puro, nenhum framework foi usado, então eu acho que será fácil para você estudá-lo se você quiser.

Como você pode ver, isto é praticamente um SPA (Single Page Application), mas você pode favoritar uma página em específico com um idioma específico atrávés de parâmetros de URL:

  • Idiomas
    • ?lang=en (Inglês. Na verdade, este é redundante porque é o padrão do site)
    • ?lang=ptbr (Português brasileiro)
  • Páginas
    • ?page=home&lang=ptbr (Página inicial. A opção padrão)
    • ?page=pro&lang=ptbr (Perfil profissional em português brasileiro)
    • ?page=projects&lang=ptbr (Projetos em português brasileiro)
    • ?page=settings&lang=ptbr (Configurações em português brasileiro)
  • Modo de iluminação
    • ?mode=light (Modo claro. Na verdade, este parâmetro é redundante porque este é o padrão)
    • ?mode=dark (Modo escuro)
  • Você já deve ter reparado no caractere &, ele nos permite usar mais de um parâmetro:
    • ?page=pro&lang=ptbr (Perfil profissional em português brasileiro)
    • ?lang=en&page=projects (Página de projetos em inglês, embora especificar "lang" como "en" seja redundante)
    • ?page=pro&lang=ptbr&mode=dark (Perfil profissional em português brasileiro e no modo escuro)
